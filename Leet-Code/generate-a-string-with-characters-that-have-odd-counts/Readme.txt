# generate-a-string-with-characters-that-have-odd-counts
https://leetcode.com/problems/generate-a-string-with-characters-that-have-odd-counts/ <br>

Example :<br>

Input: n = 4<br>
Output: "pppz"<br>
Explanation: "pppz" is a valid string since the character 'p' occurs three times and <br>
the character 'z' occurs once. Note that there are many other valid strings such as "ohhh" and "love".<br>
