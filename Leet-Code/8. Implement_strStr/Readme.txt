# 28. Implement strStr()
https://leetcode.com/problems/implement-strstr/

** About Problem : **

Given two strings needle and haystack, return the index of the first occurrence of needle in haystack, or -1 if needle is not part of haystack.<br>

** Solution : **
We can solve this problem with two approach,<br>

i) Naive pattern search algorithm.<br>
ii) KMP pattern search algorithm.<br>

My approach is first one.
